MGC Samples

Sample uses of the Maths Graphic MGC framework, a Mathematics Education Domain Specific Language, based on the AngularJS framework.

Numeric Stepper

What?

A Numeric Stepper with mouse-wheel and buttons.

Stepper value = {{amount}}

Why?

The alternative <input type="number"> does not yet have mouse wheel control or stepper buttons in all browsers.

How?

This example shows how you can use mgc.config for greater convenience. You can still pass the configuration as the expression (even in addition to mgc.config).

<span mgc-currency ng-model="nums.pos"></span>
<span ui-currency="{ symbol: '$', zero: 'my-zero-class'}" ng-model="nums.zero"></span>
<span ui-currency ng-model="nums.neg"></span>

<script>
myModule.value('ui.config', {
   currency: {
      symbol: 'USD$',
   }
});
</script>
<style>
span[ui-currency] {
  display:block;
}
.ui-currency-pos {
  font-weight: bold;
}
.ui-currency-neg {
  font-size: 125%;
}
.my-zero-class {
  color: #000;
}
</style>
